## 💎 AST TOKEN & 4IN1 POINTS  

| Parameter | AST TOKEN | 4IN1 POINTS |
|------------|------------|-------------|
| Type | Governance + Utility | Activity + Loyalty |
| Supply | 1,000,000,000 Fixed | Unlimited |
| Halving | Every 4 Years | None |
| Burn System | Active (On ASRAT-owned services only) | Auto-expiry after 12 months |
| Utility | Voting, DAO, Marketplace, NFT, Wallet | Rewards, Tips, Internal use |
| Rate | 1 AST = 1,000 4IN1 Points | Internal conversion |
| Buyable | Yes (DEX & In-app) | No (Earned only) |

**🔥 Burn Mechanism:**  
- 5–10% burn applies only to **ASRAT-owned products/services**, not user trades.  
- Voting engine includes Proof-of-Burn entry for candidates.  
- All burns visible on **AsratChain Explorer**.
